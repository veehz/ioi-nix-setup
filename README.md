# C/C++ and Python Workspace for IOI-like contests

## How to Set Up Environment

Go to [Firebase Studio](https://studio.firebase.google.com/).

Click `Import Repo`, then paste the GitHub link of this project (https://github.com/veehz/ioi-nix-setup).

It may take some time for it to build and load.

After entering, try it out! Go to your favourite language's guide:

1. [C++](/guides/cpp.md)
2. [Python](/guides/python.md)
